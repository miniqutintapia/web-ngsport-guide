# web-ngsport-guide

本仓库用于归档和发布多个独立的 HTML 页面，内容涉及网络服务相关的使用说明与参考指南。

## 项目简介

仓库以静态 HTML 页面形式存放一系列文档与指引。所有页面均可直接通过浏览器打开，无需后端服务或数据库支持。项目初衷是提供一个集中存放和分发这些页面的地方，方便用户查阅与使用。

## 目录说明

```
web-ngsport-guide/
├── index.html          # 入口页面，列出所有归档页面链接
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、图片等）
│   ├── style.css
│   └── images/
└── README.md           # 本文件
```

- **index.html**：仓库首页，可作为导航入口。
- **pages/**：所有独立 HTML 页面存放于此，每个文件对应一个独立内容。
- **assets/**：存放页面通用的样式、图片等静态资源。

## 页面归档说明

- 每个页面均为完整、自包含的 HTML 文件，可直接在浏览器中打开。
- 页面之间无依赖关系，可独立使用。
- 页面内容可能随需要更新，历史版本保留在 Git 提交记录中。
- 页面文件命名尽量简洁，反映其核心内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `index.html` 查看页面列表。
3. 点击对应链接打开具体页面，或直接在 `pages/` 目录下找到需要的 HTML 文件打开。

也可通过 GitHub Pages 等静态托管服务直接访问仓库内容，具体访问方式请参考仓库设置页面。
